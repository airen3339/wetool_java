## WeTool 工具集

一款基于`JavaFX`开发的轻量级工具集

### 有哪些功能？

- [x] 批量重命名文件，复制文件，合并文件

- [x] 监听系统剪贴板，随时查看剪贴板历史（仅支持字符串）

- [x] 查看本地IP地址，MAC地址，以及域名解析

- [x] 基于 `FastJson` 的JSON解析，可用于测试路径语法

- [x] 常见的文件编码互相转换

- [x] 二进制与十六进制的互转

- [x] 二维码生成器

- [x] 随机字符生成器

- [x] 自定义快速打开文件菜单

> 更多功能期待你的参与。。。。。。
